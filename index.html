Home Alert Messaging Program

This page describes a private, internal SMS messaging program used to deliver transactional system alerts from a Home Assistant smart home installation to a single authorized user (the system owner).

Program Details

Program name:
Home Alert Messaging
Message purpose:
Transactional system notifications only
Message types:
Security events, garage door status, water leak alerts, environmental warnings, system health notifications
Audience:
Single authorized user (system owner)
Message frequency:
Event-driven; varies (typically fewer than 5 messages per day)

Opt-In & Consent

The recipient explicitly opts in by entering their own mobile phone number during system configuration and enabling SMS alerts.

Enrollment is controlled by the user. No third-party numbers are added.

Opt-Out & Help

Reply STOP to opt out at any time.

Reply HELP for assistance.

Additional Information

Message and data rates may apply. Consent is not required to purchase any goods or services.
